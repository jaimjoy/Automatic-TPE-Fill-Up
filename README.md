# Automatic-TPE-Fill-Up
JavaScript Code that will automatically fill-up your TPE in seconds.

Step 1: Add the TPE page in **Bookmark** from any Browser.<br>
Step 2: To add, press **CTRL + D** in the TPE page. Now from Bookmark 'More' option, give your **Bookmark** a name & remove the URL, paste the **.js** code here (sometimes garbage characters will show after pasting, you've to remove them manually).<br>
Step 3: Save the Bookmark.<br>
Step 4: Now go to the TPE evaluation page & press **CTRL + SHIFT + B**.<BR>
Step 5: You will see your saved **Bookmark** in the upper section. Just click it and you're done.<br>
